# CloudWatch_server
Server side for CloudWatch_vnt

## Getting Started
Open terminal/cmd

Run:

  `git init`

  `git clone https://github.com/galigev/CloudWatch_server.git`

## Installation

Navigate to the project folder:

   `cd ../CloudWatch_server`

Run:
	
   `npm install`

## Run Server

Run:

   `nodemon`

You should get a message

"Server is up on port 3000"

Now go to the Client repository and do open the README
